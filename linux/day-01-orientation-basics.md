# Linux — Day 01: Orientation & Basic Commands

## Concept

Initial exposure to the Linux environment and basic command-line navigation.

---

## Commands Covered

### 1. `pwd` — Print Working Directory

```bash
pwd
```

```text
Displays current directory path
```

---

### 2. `cd` — Change Directory

```bash
cd /path/to/directory
cd ..
cd ~
```

```text
cd → navigate between directories
.. → move one level up
~ → home directory
```

---

### 3. `ls` — List Directory Contents

```bash
ls
ls -l
ls -a
```

```text
ls    → list files and folders
-l    → detailed view (permissions, size, owner)
-a    → include hidden files
```

---

### 4. `uname -a` — System Information

```bash
uname -a
```

```text
Displays kernel name, version, architecture
```
