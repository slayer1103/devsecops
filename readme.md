# DevSecOps Learning Repository

## Overview

This repository documents my hands-on journey learning DevSecOps from fundamentals to advanced concepts.

The goal is to move from a service desk role to a DevOps/DevSecOps role by building practical skills and structured knowledge.

---

## Structure

### Linux
Core system fundamentals:
- processes
- permissions
- users & groups
- systemd
- networking
- firewall

### Bash
Scripting and automation:
- variables, loops, conditions
- functions and arguments
- scheduling (cron, at)

### Docker
Containerization concepts:
- images and containers
- volumes and bind mounts
- networking
- Dockerfile

### Docker Compose
Multi-container orchestration:
- service definitions
- networking
- environment variables
- dependencies

### Projects
Hands-on implementations combining concepts.

---

## Learning Approach

- Concept → Practice → Documentation
- Every topic is structured for clarity and revision
- Mistakes and debugging steps are documented

---

## Goal

Build strong fundamentals and transition into a DevOps/DevSecOps role with real project experience.
